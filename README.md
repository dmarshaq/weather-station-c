# Weather Station

A weather station system that measures and records atmospheric conditions to monitor and predict weather. It provides data such as temperature, humidity, air pressure, wind, and rainfall, which are essential for forecasting, research, and daily decision-making.

Quick Start
-----------------
#### Requirements
* Git
* GCC - (GNU C Compiler)
* C Standard Library
* Linux Environment
* SDL2 Core and TTF Libraries



#### Bootstrap

1. Open a terminal and navigate to your projects directory.
2. Clone the repository:
    ```bash
    git clone https://github.com/dmarshaq/weather-station-c.git
    ```
3. Enter the project.
    ```bash
    cd weather-station-c
    ```
4. Compile the builder program.
   ```bash
   gcc -o nob nob.c
   ```
5. Build the project.
    ```bash
    ./nob
    ```
6. Run the main.
    ```bash
    bin/main
    ```

    
#### Rebuild and Run

1. To rebuild simply call builder program.
    ```bash
    ./nob
    ```

2. Run the main.
    ```bash
    bin/main
    ```
